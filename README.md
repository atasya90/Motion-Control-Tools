# Kling Motion Control Tool

Versi fix tanpa Tailwind, sudah termasuk `lib/models.ts` dan import TypeScript yang benar.

## Env Vercel

Isi di Vercel > Project > Settings > Environment Variables:

```env
FAL_KEY=isi_key_fal_ai
REPLICATE_API_TOKEN=isi_token_replicate
ADMIN_PASSWORD=password_admin_sendiri
```

## Deploy

1. Extract zip.
2. Upload semua isi folder ke GitHub.
3. Import repo ke Vercel.
4. Isi env.
5. Deploy.

## Local

```bash
npm install
npm run dev
```
